Complexity Analysis (T(n) y O(n))

T(n) of MergeSort depends on the T1(n)of merge function, the
middle point calculation, the half array assignment and the
recursive call. The merge function T1(n) can be obtained
intuitively as T(n) = 2*min(r,l) + (max(r,l)-min(r,l)), that
is: First while of two instructions executes until the end of
the smallest list is reached, then there are assignment instructions
of weight max-min length. Then: T1(n) = max(r,l)+min(r,l) = n of each
recursive operation. (Continue next page)
